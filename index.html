Universal Deployment Guide

Install Any OS in 6 Steps

This guide works for Windows, Ubuntu, Linux Mint, and every major distribution. Follow it exactly and you cannot go wrong.

01

Download the Official ISO File

Choose your OS from the cards above and click Download. You'll be redirected to the official vendor website. Download the ISO file — typically 1–5 GB depending on the OS. Never download from unofficial mirrors.

Always verify the SHA256 checksum after downloading
02

Back Up Your Existing Files First

A clean install will erase the target drive. Before you do anything else, copy your photos, documents, and important files to an external drive or cloud storage. Do not skip this step.

03

Flash ISO to a USB Drive (8 GB+)

Use a USB 3.0 drive for best speed. Flash the ISO using one of these tools — they handle all partitioning automatically. Never just copy-paste the ISO file to the USB; you must flash it properly.

Rufus (Windows) balenaEtcher (Mac/Linux) Ventoy (Multi-boot)
04

Enter BIOS / UEFI and Set Boot Order

Restart your computer and immediately press your BIOS key. Set USB as the first boot device. If installing Linux, also disable Secure Boot in the BIOS security settings.

Common BIOS keys: F2 · F12 · DEL · ESC · F10
05

Run the OS Installer

Boot from USB and follow the graphical installer. When asked about disk partitioning, choose "Erase disk and install" for a clean fresh start. The installer will guide you through language, timezone, and user account setup.

06

First Boot — Update and Configure

Installation takes 10–30 minutes. When complete, remove the USB drive and restart. After first boot, run system updates immediately. For Linux, install your GPU drivers and any missing codecs.

Ubuntu/Mint: sudo apt update && sudo apt upgrade -y
